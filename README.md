# Git Lesson

## リモートリポジトリとローカルリポジトリとはそれぞれ何でしょうか？

* リモートリポジトリとは複数人で共有するためのディレクトリやファイルの状態を管理する場所
* ローカルリポジトリとはPC上に配置するためのディレクトリやファイルの状態を管理する場所

## プッシュとマージの違いは何でしょうか？

* プッシュはローカルリポジトリからリモートリポジトリに公開するためにアップロードすること
マージは別のブランチの変更をブランチに取り込むこと

## コミットとプッシュの違い

* コミットはリポジトリに編集したファイルを保存すること
* プッシュはローカルリポジトリからリモートリポジトリに公開するためにアップロードすること

## コミットのメッセージはどのように書いてあげるのが最適でしょうか？

* どんな編集をしたのかをわかりやすく記述

## ローカルでマージするフローと、プルリクエストでマージするフローの違いは何でしょうか？

* ローカルでマージするフローだと、コードをレビューする前に反映されてしまうため、コードにバグがあっても気づきづらい。プルリクエストで自分以外の人にレビューしてもらう手順を踏めばバグを発見することができる

## コンフリクトを起こしてしまった場合、どう対処すべきですか？

1. 先にマージした内容を取り込む
2. 後にマージしようとした内容を取り込む
3. どちらの内容も取り込む
